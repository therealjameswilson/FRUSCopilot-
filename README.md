FRUS Compiler Copilot

An AI research assistant for identifying documents relevant to new
Foreign Relations of the United States (FRUS) volumes.

Features:
- Parse FRUS XML volumes
- Semantic search across FRUS corpus
- Suggest documents for new volumes
- Identify declassified government sources
- Recommend archival collections

Run locally:

pip install -r requirements.txt
python agents/volume_suggester.py
